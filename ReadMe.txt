To run the code we need Python3.

All required libraries are included in the requirements.txt.
To setup code simply run "pip install -r requirements.txt"

Running Code:
To run code simply run the Experiments.py file using:

            python Experiments.py

The results are stored in the Results.txt file.
The configurations of all the Experiments i.e. the different s,t selections for every pair is stored in the Tests.txt file.
